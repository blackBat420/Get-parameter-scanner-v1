# Get-parameter-scanner-v1
Scan all get parameters of a website in realtime while you are browsing the website
HOW TO USE

1. INSTALL TAMPER MONKEY EXTENSION ON YOUR BROWSER
2. ONCE INSTALLED, GO TO EXTENAION TAB OF YOUR BROWSER AND OPEN THE TAMPER MONKEY
3. CLICK CREATE NEW USER SCRIPTS
4. DELETE ALL PRE-LOADED SCRIPTS ON THE EDITOR AND PASTE THE SCRIPTS OF GET-PARAMETERS SCANNER.
5. SAVE THE SCRIPTBAND ENABLE
6. BROWSE ANY WEBSITE TO ACTIVATE THE SCANNER, IT SHOULD DISPLAY THE "CLOWN" BUBLE BUTTON TO SHOW THE SCANNED RESULTS
